# Installation

## For Admin System

* Go to Web panel of the Admin

* Sign in using Admin credentials

* Download the required Archived files

* Extract the downloaded folder

* Open Admin.zip

* Run Setup.sh on the required system

## For Client System

* Go to Web panel of the Admin

* Sign in using Admin credentials

* Download the required Archived files

* Extract the downloaded folder

* Open Client.zip

* Run Setup.sh on the required system
